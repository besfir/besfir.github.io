How to measure execution time in Java/Kotlin. +Spring Framework

Besfir · June 8, 2023

System.currentTimeMillis() and System.nanoTime() are widely known as popular methods for measuring run time in Java. In this article, we will measure the execution time using the latter for several reasons, and we will briefly summarize the reason.

Why System.nanoTime()?

First, as the name suggests, it can be measured up to nanoseconds, so it can be measured 106 times more precisely.

Second, if you read Java doc, you can see the explanation, This method can only be used to measure elapsed time and is not related to any other notion of system or wall-clock time.. As can be seen from the description, this method was created from the beginning to measure elapsed time.

Third, System.currentTimeMillis() returns wall-clock time, so inaccurate values can be measured by external variables such as OS time changes or leap seconds.

Java

long start = System.nanoTime();
// Heavy task simulation
Thread.sleep(500);
long finish = System.nanoTime();
// via milliseconds
long elapsedTime = (finish - start) / 1_000_000;
System.out.printf("elapsed = %sms" , elapsedTime);
Output
elapsed = 505ms

Kotlin

You can use Java code as it is, but let's use it because it provides a util method that makes it easier to measure execution time in Kotlin.

measureTimeMillis()

The execution time can be measured in milliseconds.

import kotlin.system.measureTimeMillis

val elapsed = measureTimeMillis {
  // Heavy task simulation
  Thread.sleep(500)
}
println("elapsed = ${elapsed}ms")
Output
elapsed = 507ms

measureNanoTime()

The execution time can be measured in nanoseconds.

import kotlin.system.measureNanoTime

val elapsed = measureNanoTime {
  // Heavy task simulation
  Thread.sleep(500)
}
println("elapsed = ${elapsed}ns")
Output
elapsed = 504509000ns

measureTime()

Returns the Duration object whose value has been converted from nanoseconds to milliseconds.

import kotlin.time.measureTime

val elapsed = measureTime {
  // Heavy task simulation
  Thread.sleep(500)
}
println("elapsed = $elapsed")
Output
elapsed = 505.351542ms

measureTimedValue()

Returns the TimedValue data class consisting of measurement time and in-block return values.

import kotlin.time.measureTimedValue

val timedValue = measureTimedValue {
  // Heavy task simulation
  Thread.sleep(500)
  "Hello world"
}
println("timedValue = $timedValue")
Output
timedValue = TimedValue(value=Hello world, duration=507.327875ms)

Spring framework

Get total time

import org.springframework.util.StopWatch;

StopWatch sw = new StopWatch();

sw.start();
//Heavy task simulation
Thread.sleep(200);
sw.stop();

sw.start();
//Heavy task simulation
Thread.sleep(500);
sw.stop();

sw.start();
//Heavy task simulation
Thread.sleep(1000);
sw.stop();

System.out.printf("elapsed = %sms" , sw.getTotalTimeMillis());
Output
elapsed = 1714ms

Get last task time

import org.springframework.util.StopWatch;

StopWatch sw = new StopWatch();

sw.start();
//Heavy task simulation
Thread.sleep(200);
sw.stop();

sw.start();
//Heavy task simulation
Thread.sleep(500);
sw.stop();

sw.start();
//Heavy task simulation
Thread.sleep(1000);
sw.stop();

System.out.printf("elapsed = %sms" , sw.getLastTaskTimeMillis());
Output
elapsed = 1005ms

Get summary by task name

import org.springframework.util.StopWatch;

StopWatch sw = new StopWatch();

sw.start("A");//setting a task name
//Heavy task simulation
Thread.sleep(200);
sw.stop();

sw.start("B");//setting a task name
//Heavy task simulation
Thread.sleep(500);
sw.stop();

sw.start("C");//setting a task name
//Heavy task simulation
Thread.sleep(1000);
sw.stop();

System.out.println(sw);
System.out.println();
System.out.println(sw.prettyPrint());
Output

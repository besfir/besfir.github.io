Intl RelativeTimeFormat으로 timeago.js와 비슷한 커스텀 함수만들기.

Besfir · 2023년 5월 29일
목차
  • 코드
  • 해설

인스타그램, 트위터, 유튜브 등을 보면 xx 초 전, xx 분 전과 같은 상대시간 형태로 컨텐츠가 올라온 시간이 표시된다. 라이브러리를 사용하면 쉽게 구현이 가능하지만, JS 내장 API인 Intl을 통해 쉽고 비슷하게 구현할 수 있기 때문에 이번 기회에 정리해보고자 한다.

코드

해설

상대 시간으로 표현하고자 하는 단위와 ms 단위로 환산한 값을 짝지어 미리 선언했다.

const units = {
    year: 31_536_000_000,   // 864_000_000(day) * 365
    month: 2_629_800_000,   // 31_536_000_000(year) / 12
    week: 604_800_000,      // 86_400_000(day) * 7 
    day: 86_400_000,
    hour: 3_600_000,
    minute: 60_000,
    second: 1_000,
}

모든 단위를 순회하면서 각 단위가 1 이상으로 나눠지는지 확인한다. 0 이상이 나온다면, 그 단위에 해당하지 않으므로 다음 단위로 넘어간다.

ex) diff = 120_123, amount = 2, unit = 'minute'

function formatRelative(when, locale = 'en') {
    const formatter = new Intl.RelativeTimeFormat(locale, {
        numeric: "auto",
    });
    const diff = when - Date.now();
    for (const [unit, value] of Object.entries(units)) {
        const amount = Math.trunc(diff / value);        //(1)
        if (amount || unit === 'second') {              //(2)
            return formatter.format(amount, unit);
        }
    }
}

(1) 소수점을 절삭하기 위해 Math.trunc()를 사용했다. 표현 구조상 -1_123ms는 1초 전으로, 1_123ms는 1초 후로 표기되기 때문에 0에 가깝도록 표기해야 하므로 소수점을 절삭이 필요했다.

(2) 1초 미만의 값들은 unit==='second'에서 걸러져서 지금을 반환한다.

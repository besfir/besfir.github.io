Java/Kotlin에서 실행 시간 측정 하기. +Spring Framework

Besfir · 2023년 6월 8일
목차

  • Java
  • Kotlin
  • measureTimeMillis()
  • measureNanoTime()
  • measureTime()
  • measureTimedValue()
  • Spring framework

자바에서 실행 시간을 측정하는 대중적인 방법으로 System.currentTimeMillis()와 System.nanoTime()이 많이 알려져있다. 이 글에서는 몇가지 이유로 후자를 사용해서 실행 시간을 측정을 할 예정인데, 그 이유도 간단히 정리해보도록 하겠다.

왜 System.nanoTime()을 선택했을까?

첫째로, 이름에서 알 수 있듯이 나노초 단위까지 측정할 수 있기 때문에 106배 정밀하게 측정할 수 있다.

둘째로, Java doc을 읽어보면 This method can only be used to measure elapsed time and is not related to any other notion of system or wall-clock time. 라는 설명을 볼 수 있다. 설명을 통해 알 수 있듯이, 이 메서드는 애초부터 경과시간 측정을 위해 만들어진 메서드이다.

셋째는 System.currentTimeMillis()는 wall-clock time을 반환하기 때문에 OS의 시간 변경이나 윤초와 같은 외부 변수에 의해 부정확한 값이 측정될 수 있다.

Java

long start = System.nanoTime();
// Heavy task simulation
Thread.sleep(500);
long finish = System.nanoTime();
// via milliseconds
long elapsedTime = (finish - start) / 1_000_000;
System.out.printf("elapsed = %sms" , elapsedTime);
Output
elapsed = 505ms

Kotlin

자바 코드를 그대로 사용해도 되지만, Kotlin에서 더 쉽게 실행시간을 측정할 수 있게 해주는 유틸함수를 제공하기 때문에 이를 사용해보겠다.

measureTimeMillis()

밀리초 단위로 실행 시간을 측정할 수 있다.

import kotlin.system.measureTimeMillis
val elapsed = measureTimeMillis {
  // Heavy task simulation
  Thread.sleep(500)
}
println("elapsed = ${elapsed}ms")
Output
elapsed = 507ms

measureNanoTime()

나노초 단위로 실행 시간을 측정할 수 있다.

import kotlin.system.measureNanoTime
val elapsed = measureNanoTime {
  // Heavy task simulation
  Thread.sleep(500)
}
println("elapsed = ${elapsed}ns")
Output
elapsed = 504509000ns

measureTime()

Duration객체를 반환하며, 나노초 단위를 밀리초 단위로 환산한 값을 가진다.

import kotlin.time.measureTime
val elapsed = measureTime {
  // Heavy task simulation
  Thread.sleep(500)
}
println("elapsed = $elapsed")
Output
elapsed = 505.351542ms

measureTimedValue()

측정시간과, 블록 내 반환값으로 구성된 TimedValue 데이터 클래스를 반환한다.

import kotlin.time.measureTimedValue
val timedValue = measureTimedValue {
  // Heavy task simulation
  Thread.sleep(500)
  "Hello world"
}
println("timedValue = $timedValue")
Output
timedValue = TimedValue(value=Hello world, duration=507.327875ms)

Spring framework

전체 시간 출력

import org.springframework.util.StopWatch;
StopWatch sw = new StopWatch();
sw.start();
//Heavy task simulation
Thread.sleep(200);
sw.stop();
sw.start();
//Heavy task simulation
Thread.sleep(500);
sw.stop();
sw.start();
//Heavy task simulation
Thread.sleep(1000);
sw.stop();
System.out.printf("elapsed = %sms" , sw.getTotalTimeMillis());
Output
elapsed = 1714ms

마지막 작업만 출력

import org.springframework.util.StopWatch;
StopWatch sw = new StopWatch();
sw.start();
//Heavy task simulation
Thread.sleep(200);
sw.stop();
sw.start();
//Heavy task simulation
Thread.sleep(500);
sw.stop();
sw.start();
//Heavy task simulation
Thread.sleep(1000);
sw.stop();
System.out.printf("elapsed = %sms" , sw.getLastTaskTimeMillis());
Output
elapsed = 1005ms

구간별 측정

import org.springframework.util.StopWatch;
StopWatch sw = new StopWatch();
sw.start("A");//setting a task name
//Heavy task simulation
Thread.sleep(200);
sw.stop();
sw.start("B");//setting a task name
//Heavy task simulation
Thread.sleep(500);
sw.stop();
sw.start("C");//setting a task name
//Heavy task simulation
Thread.sleep(1000);
sw.stop();
System.out.println(sw);
System.out.println();
System.out.println(sw.prettyPrint());
Output
